# AI Resume Analyzer 🚀

Live Demo: https://ai-resume-analyzer-nu-smoky.vercel.app/  
Backend API: [https://your-render-link.onrender.com ](https://ai-resume-analyzer-uptg.onrender.com) 

## Features
- Resume upload & parsing
- AI-powered analysis
- Score breakdown
- Job description match
- Bullet point improver

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- AI: Groq (LLaMA 3.3)
- Database: MongoDB
